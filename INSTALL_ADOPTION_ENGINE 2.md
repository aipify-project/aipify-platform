# Install & Adoption Engine (ABOS)

**Feature owner:** Customer App  
**Implementation:** [Aipify Install Engine — Phase A.22](./AIPIFY_INSTALL_ENGINE_PHASE_A22.md) · [Install Engine — Phase 17](./INSTALL_ENGINE.md)

Aipify should adapt to organizations. Organizations should not have to rebuild themselves around Aipify.

## Mission

Reduce friction during onboarding and accelerate time-to-value through intelligent installation, discovery, and adoption experiences.

## Core philosophy

The easiest systems to adopt are the systems that understand their environment. Aipify integrates into existing workflows respectfully and intelligently.

## Installation principles

- Meet organizations where they are
- Learn existing structures
- Respect existing processes
- Recommend improvements gradually
- Avoid overwhelming new users

## Supported platforms

WordPress · Shopify · WooCommerce · Custom websites · Future business platforms

## Adoption journey

| Stage | Focus |
|-------|--------|
| **Welcome** | Introduce Aipify, explain purpose, establish trust |
| **Discovery** | Learn the organization, understand teams, identify priorities |
| **Assistance** | Begin helping, recommend improvements, reduce friction |
| **Partnership** | Embedded in operations, support growth, preserve organizational memory |

## ABOS principle

> Technology should adapt to people. People should not be forced to adapt entirely to technology.

## Vision

Adoption should feel less like installation and more like welcoming a new companion to the team — *"It understood us. It helped us quickly. It became part of how we work."*

## Routes

- **Install & Adoption dashboard:** `/app/aipify-install-engine`
- **Modern install wizard:** `/app/install`
- **Embedded runtime:** `/api/install/*`, `/api/embed/*`

## Dogfooding

Aipify Group AS validates installation internally. Unonight is the first external pilot.

## Distinctions

Extends **Install Engine (Phase 17)** — not a duplicate. Integrates **Customer Onboarding Engine**, **Knowledge Center (A.5)**, and **Organizational Memory (A.34)** on completion.
