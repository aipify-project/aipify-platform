# Companion Memory Expansion Engine — FAQ

## What is Companion Memory?

Companion Memory allows Aipify to remember approved information so assistance becomes more personalized and useful over time.

## Can I control what Aipify remembers?

Yes. Memory remains transparent and manageable through the Memory Review Center at `/app/companion/memory`.

## Can memories be deleted?

Yes. Authorized users may review, edit, archive or delete memories.

## Does Aipify remember everything automatically?

No. Suggested memories require review. Low-confidence memories are never treated as facts.
