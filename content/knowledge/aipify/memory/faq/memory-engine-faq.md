---
title: What information does Aipify store?
slug: what-information-does-memory-store
category: memory-engine
language: en
visibility: authenticated
status: published
tags: [memory, privacy, data]
article_type: faq
---
Memory Engine stores user preferences (language, notification modes, briefing habits), team patterns (approval chains, support routines), and tenant-level trends (common incidents, knowledge gaps, integration behaviours). All memories include explanations where possible.

---
title: What information does Aipify never store?
slug: what-information-does-memory-never-store
category: memory-engine
language: en
visibility: authenticated
status: published
tags: [memory, privacy, security]
article_type: faq
---
Aipify never stores passwords, API secrets, sensitive personal health data, payment details, or memory shared across tenants. Sensitive keys are filtered before any observation is saved.
