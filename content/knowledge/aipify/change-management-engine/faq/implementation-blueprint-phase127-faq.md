# Implementation Blueprint Phase 127 — Transformation Orchestration & Change Companion FAQ

## What is Phase 127?

Phase 127 extends the Change Management Engine (Phase A.47 + Blueprint Phase 62) with **Enterprise Intelligence transformation orchestration** — roadmap scaffolds, change companion support, adoption intelligence, transformation memory, and communication orchestration at `/app/change-management-engine`.

## How is Phase 127 different from Phase 62?

**Phase 62** adds people-centered change framing via `_cmbp_*`. **Phase 127** adds `_tcobp127_*` orchestration center, transformation roadmap engine, adoption intelligence, and transformation memory — all Phase 62 fields are preserved.

## How is this different from Evolution Governance Phase 84?

**Evolution Governance Phase 84** at `/app/evolution` manages **Aipify software evolution proposals**. **Phase 127** orchestrates **organizational transformation initiatives** inside the customer tenant.

## Does Phase 127 create a new route?

No. Phase 127 extends the existing `/app/change-management-engine` dashboard and card RPCs.

## What helper prefixes are used?

Engine: `_cme_*` (A.47). Blueprint Phase 62: `_cmbp_*`. Blueprint Phase 127: `_tcobp127_*` — must not collide.

## How does adoption intelligence work?

Seven aggregate indicators (training participation, companion usage, process adoption, etc.) — **support not surveillance**. No individual employee ranking or punitive framing.

## How does Organizational Memory Phase 126 relate?

Phase 127 captures transformation memory metadata (success factors, challenges, outcome reviews) and cross-links `/app/organizational-memory-engine` — does not duplicate OME RPCs.

## What are Change Companion limitations?

Never force participation · Never manipulate emotions · Never suppress concerns · Never replace accountability · Never guarantee success.

## What cross-links are mandatory?

Executive Intelligence 121 · Decision Intelligence 125 · Digital Twin 124 · Org Memory 126 · Aipify University 115 · Self Love A.76 · Learning & Training A.36 · Stakeholder Communication A.53.

## Where is the dashboard?

`/app/change-management-engine` — `get_change_management_engine_dashboard()` returns all Phase A.47, Phase 62, and Phase 127 fields.
