# Global Knowledge Preservation & Civilizational Memory — Phase 163 FAQ

## What is Phase 163?

Post-Enterprise & Civilizational Phase 163 introduces the **Civilizational Memory Center** at `/app/civilizational-memory-engine` for intentional knowledge preservation with discernment.

## Does Phase 163 preserve everything?

**No.** Preservation requires discernment — not everything deserves archival. Phase 163 explicitly avoids digital clutter and hoarding. Wisdom curation reviews identify essential knowledge, outdated knowledge, and identity-shaping stories worth stewarding.

## Does Phase 163 rewrite history?

**No.** The Memory Companion supports remembrance — it does **not** alter historical records, suppress alternative experiences, determine truth, replace institutional judgment, or improperly reveal confidential information.

## How does Phase 163 relate to Global Knowledge Exchange Phase 141?

Phase 141 at `/app/global-knowledge-exchange-engine` focuses on **exchange** — cross-organizational learning. Phase 163 focuses on **preservation** with discernment. Cross-link only — never duplicate `_gkee_*` / `_gkeebp141_*` RPCs.

## How does Phase 163 relate to Organizational Memory Phase 152?

Phase 152 at `/app/organizational-memory-engine` handles tenant legacy and succession intelligence. Phase 163 handles civilizational memory and wisdom curation at a broader stewardship layer. Cross-link only — never duplicate `_olsibp152_*` / `_om_*` RPCs.

## How does Phase 163 relate to Living Enterprise Phase 160?

Phase 160 at `/app/living-enterprise-engine` provides living memory and transcendence visibility. Phase 163 cross-links living memory themes — it does not duplicate Phase 160 RPCs.

## How does Phase 163 relate to Legacy Engine A.86?

Full legacy stories live in **Legacy Engine A.86** at `/app/legacy-engine`. Phase 163 stores metadata scaffolds and cross-link counts only — never duplicates `_leg_*` tables or RPCs.

## What does the Memory Companion do?

The Memory Companion provides knowledge summaries, wisdom retrieval, historical context, reflection prompts, legacy preparation, and learning recommendations. It does **not** alter records, suppress experiences, determine truth, replace judgment, or improperly reveal confidential info.

## What data does Phase 163 store?

Metadata summaries (max ~500 chars) and aggregate counts only. Governed retention. No raw chat, email, PII, or confidential records in default RPC payloads.

## Which engines does Phase 163 cross-link?

Civic Collaboration Phase 161, Cross-Sector Phase 162, Global Knowledge Exchange Phase 141, Organizational Memory Phase 152, Legacy A.86, Decision Heritage Phase 153, Future Leaders Phase 151, Living Enterprise Phase 160, Knowledge Center A.5, and Self Love A.76.

## What thin RPCs are available?

- `archive_civilizational_memory_entry(...)` — archive wisdom/lesson metadata
- `record_wisdom_curation_review(...)` — record curation/review cycle metadata

## What permissions are required?

- `civilizational_memory.view` — view archives and dashboards
- `civilizational_memory.manage` — manage preservation settings
- `civilizational_memory.steward` — conduct wisdom curation reviews
