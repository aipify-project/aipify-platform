# Platform Installation — FAQ

## Can anyone install Aipify?

Yes. Aipify is designed so that both technical and non-technical users can install it with guided help from the Aipify Install Assistant.

## Which platforms can Aipify be installed on?

Aipify supports WordPress, Shopify, WooCommerce and other platforms through plugins, apps, scripts, APIs and webhooks.

## Do I need to register a payment method to start the trial?

Yes. A payment method is required to activate the 14-day trial, but you are not charged when the trial begins.

## Will I be charged immediately?

No. You will not be charged at registration. The first charge happens only after the 14-day trial is over, unless you cancel before the trial ends.

## Can I cancel before the trial ends?

Yes. If you cancel before the trial ends, no charge is made.

## Why does Aipify require payment information for a free trial?

Payment registration confirms serious usage and allows the subscription to continue automatically after the trial without interrupting service.

## Can Aipify help me install itself?

Yes. Aipify guides the customer through installation, verifies each step and explains how to fix any issues.

## What happens if installation fails?

Aipify will explain what failed, why it matters and how to fix the problem. If needed, the issue can be escalated to support.

## Can Aipify verify that installation worked?

Yes. Aipify runs a health check to verify connection, permissions, widget visibility, billing status and module activation.

## Can developers install Aipify manually?

Yes. For custom platforms, Aipify provides API keys, webhook configuration, JavaScript embeds and developer documentation.

## Can agencies install Aipify for customers?

Yes. Certified partners and agencies may help customers install and configure Aipify.

## What happens after the 14-day trial?

If the customer has not cancelled, the selected subscription starts automatically and the first payment is charged.

## Can I change plan during the trial?

Yes. Customers can upgrade or change plan during the trial. Billing still begins only after the trial period ends.
