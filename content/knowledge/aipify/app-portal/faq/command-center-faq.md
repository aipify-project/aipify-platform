# What is the Command Center?

The Command Center provides leaders with a unified view of organizational priorities, execution, risks and opportunities.

# Does the Command Center replace individual modules?

No. The Command Center summarizes information from across Aipify while allowing users to drill down into specialized areas when needed.

# Can Aipify run the organization automatically?

No. Aipify provides insights, recommendations and awareness while organizational leaders remain responsible for decisions and execution.
