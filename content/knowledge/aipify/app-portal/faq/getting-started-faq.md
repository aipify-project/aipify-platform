# Getting Started — APP Portal Knowledge Center FAQ

## What is Getting Started?

Getting Started helps organizations configure Aipify correctly and accelerate adoption through guided onboarding.

## Do I need to complete every onboarding step?

Some onboarding steps are optional, while others help improve security, collaboration and long-term success.

## Can onboarding be revisited later?

Yes. Organizations can return to Getting Started at any time to complete remaining setup tasks.
