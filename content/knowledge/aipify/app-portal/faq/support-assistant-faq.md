# Support Assistant — APP Portal Knowledge Center FAQ

## What is Aipify Support Assistant?

Aipify Support Assistant helps customers find answers from the Knowledge Center before creating a support request.

## Does Support Assistant replace human support?

**No.** The Support Assistant helps users find guidance faster, but customers can still create support requests when human assistance is needed.

## Can Support Assistant create a support request?

The Support Assistant can help prepare support request context, but the user must confirm before a request is created.
