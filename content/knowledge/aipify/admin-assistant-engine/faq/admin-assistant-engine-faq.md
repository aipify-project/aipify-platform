# Admin Assistant Engine — FAQ

## What is the Admin Assistant?

The Admin Assistant is an Aipify-powered helper designed to support daily administrative activities.

## Can the Admin Assistant perform actions automatically?

Only approved low-risk actions may execute automatically.

## What is the Since Last Login feature?

It provides a summary of important events that occurred since the user's previous session.

## Can the Admin Assistant assign tasks?

Yes. Tasks may be created and assigned according to permissions and approval policies.

## How does the assistant prioritize recommendations?

Recommendations consider urgency, business impact, and available organizational knowledge.

## Are assistant actions audited?

Yes. Important assistant activities are recorded in audit logs.
