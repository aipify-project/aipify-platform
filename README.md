# aipify-platform

The core platform for Aipify AI.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) to view the landing page.

## Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run lint` — run ESLint

## i18n

Supported locales: `en`, `no`, `sv`, `da`. Translation files live in `locales/`.
